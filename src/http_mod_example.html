handle/4

handle(Socket, Http_request::Request, Response, Flags) -> {proceed, NewResponse, NewFlags} | {skip_to, Module, NewResponse, NewFlags} | Response | already_sent | {error, closed} | {error, Reason}

  • Socket = http_socket()
  • Request = #http_request{}
  • Response = #http_response{}
  • Flags = proplist()
  • NewResponse = #http_response{}
  • NewFlags = proplist()
  • Module = atom()
  • Reason = string()

The entry point to the module.

Returns one of the following:
{proceed, Response, Flags}
Pass the Response and the Flags to the next module in the modules list. If this was the last module, send the Response to the client socket.
{skip_to, Module, Response, Flags}
Skip to the specified Module. If Module is not found, send the Response to the client socket.
#http_response{}
Modules traversal is complete. Response will be sent to the socket.
already_sent
Modules traversal is complete. The response has already been sent to the client socket.
{error, closed}
Client socket was closed. Exit the handler.
{error, Reason::string()}
Internal Server Error (500) will be sent to the client. The error Reason will be logged.
